# Leetcode

## 记录

  用于记录自己毕业时候做的相关leetcode算法题
  oj是属于自己希望能通过ACM模式来锻炼自己不仅仅只是写相关算法接口，
  更多的是调试自己的代码能力
  以及有一个从网上摘抄来的C++知识点的总结，时刻提醒自己，基础知识的重要性。

### 毕业相关
  拖延一个多月没有刷题目是忙于毕业设计，现在因为已经毕业了，再加上自己决定二战考研，所以7月1日开始，开leetcode会员进行强化性的刷题目，理解对于我们这种双非院校的普通大学生来说，贵在坚持刷算法，一直坚持学习，坚持反思自己的阶段性的功过，可能有那么一段时间是停滞不前的，但是人总是走在属于自己的道路上，愿自己二战圆梦，能力也要提升一大截，加油！！！！

### 警示

  学习如逆水行舟，不进则退，也是只有不断的学习自己，强化自己的能力，多读一些能够切实提升自己的感官能力，毛选、曾国藩家书等能够切实提升自己的修养，人格上的魅力的书。
